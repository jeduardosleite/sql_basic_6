#### **Tables**

##### transacoes
```sql

    id_cliente BIGINT,
    id_transacao BIGINT,
    valor_compra DOUBLE,

)
ROW FORMAT SERDE 'org.apache.hadoop.hive.serde2.lazy.LazySimpleSerDe'
WITH SERDEPROPERTIES (
  'serialization.format' = ',',
  'field.delim' = ','
)
LOCATION 's3://mod6-jeduardo/transacoes/';
```
